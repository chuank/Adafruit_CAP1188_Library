This is an Arduino library for the CAP1188 8-channel Capacitive touch sensor. It also works with Particle family of microcontrollers.

It extends the original Adafruit library to allow customisation of the astoundingly-configurable CAP1188.

Refer to Adafruit_CAP1188.h header to review new functions exposed.

References to register address in the datasheet can also be found there.

**** THIS LIBRARY ONLY IMPLEMENTS IC2 PROTOCOL ****



Original library:
https://github.com/adafruit/Adafruit_CAP1188_Library
